UnDone

  • CSS/SASS
  • Bootstrap
  • Ruby on Rails
  • Devise (Authentication)

UnDone is a to-do list application, built with Ruby on Rails that consists of 3 models (users, tasklists, and tasks). Users are required to sign up or sign in to use the application and the authentication is handled by Ruby's Devise gem. Once signed in users have the ability to create multiple tasklists and add, delete, or complete tasks from the lists.
